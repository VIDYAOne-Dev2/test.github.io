Before You Continue

System
Prerequisites

To ensure a smooth installation and reliable performance, please take a moment to confirm that your system meets the basic requirements below. Most modern school computers already do.

Need help checking this?

Operating System

Latest version of Windows installed (Windows 10 or Windows 11 recommended).

.NET Framework

Microsoft .NET Framework version 4.6 or above installed.

Processor & Memory

Intel i3 processor or higher with at least 4GB RAM.

Internet Connection

A stable internet connection is required only to download the software setup.

Quick System Check

Security & System Settings

Before installation, please review a few simple system settings. These checks help avoid interruptions and ensure VIDYAOne runs smoothly from day one.

Recommended

Windows Firewall

Ensure Windows Firewall allows VIDYAOne to run without restriction. Adding the application to allowed apps prevents unnecessary blocking.

Required

Antivirus Protection

Keep an active antivirus installed. If needed, add VIDYAOne to the antivirus exclusion list to avoid false alerts.

Important

Date & Time

Confirm your system date, time, and time zone are correct. Automatic time sync is recommended for accurate records.

Access

Folder Permissions

Make sure the VIDYAOne installation folder has full read and write access. This ensures smooth data storage and report generation.
